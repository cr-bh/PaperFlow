# 功能演示截图说明

本目录用于存放 PaperBrain 的功能演示截图。

## 📸 需要的截图

请按照以下命名规范添加截图：

### 1. dashboard.png
**功能**: 论文列表页面
**内容要求**:
- 显示论文卡片列表
- 包含论文标题、作者、标签
- 展示搜索和筛选功能

**推荐尺寸**: 1200x800 像素

---

### 2. paper-detail.png
**功能**: 论文详情页 - 结构化笔记
**内容要求**:
- 显示 8 维度结构化总结
- 包含问题定义、方法论、实验结果等部分
- 展示笔记编辑功能

**推荐尺寸**: 1200x800 像素

---

### 3. mindmap.png
**功能**: 思维导图
**内容要求**:
- 显示完整的论文思维导图
- 展示 Mermaid.js 渲染效果
- 包含不同颜色的节点

**推荐尺寸**: 1200x800 像素

---

### 4. auto-scholar.png
**功能**: Auto-Scholar 论文监控
**内容要求**:
- 显示论文抓取列表
- 包含 S/A/B 分级标签
- 展示会议徽章和机构徽章
- 显示评分和推荐理由

**推荐尺寸**: 1200x800 像素

---

## 📝 截图步骤

1. **启动应用**:
   ```bash
   streamlit run app.py
   ```

2. **准备数据**:
   - 上传几篇论文
   - 运行 Auto-Scholar 抓取
   - 确保有完整的数据展示

3. **截图工具**:
   - macOS: `Cmd + Shift + 4` 选择区域截图
   - Windows: `Win + Shift + S` 截图工具
   - 或使用浏览器开发者工具截图

4. **保存截图**:
   - 将截图保存到 `docs/images/` 目录
   - 使用上述指定的文件名
   - 格式: PNG（推荐）或 JPG

5. **验证显示**:
   - 在 README.md 中查看图片是否正常显示
   - 确保图片清晰、内容完整
